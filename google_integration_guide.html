دليل ربط البرنامج بأدوات جوجل (Google & Firebase)

دليل إعداد وتشغيل تطبيق مبيعات لوحات الطرق أونلاين مجاناً

الجزء الأول: إنشاء مشروع سحابي على جوجل (Google Firebase)

  1. قم بزيارة موقع Firebase Console وسجل الدخول بحساب الجيميل الخاص بك.
  2. اضغط على زر Create a project (إنشاء مشروع).
  3. اكتب اسماً للمشروع (مثال: HT-Advertising-Suez) ثم اضغط Continue.
  4. يمكنك تفعيل أو إلغاء تفعيل Google Analytics (حسب رغبتك) ثم اضغط Create Project.
  5. انتظر ثوانٍ حتى يتم تهيئة خوادم جوجل، ثم اضغط Continue لتفتح لك لوحة التحكم السحابية.

الجزء الثاني: تفعيل تسجيل الدخول بجوجل (Google Sign-In)

  1. من القائمة الجانبية اليسرى في لوحة Firebase، اضغط على Build ثم اختر Authentication (المصادقة).
  2. اضغط على زر Get Started للبدء.
  3. من تبويب Sign-in method، اختر مقدم الخدمة Google.
  4. قم بتشغيل خيار التفعيل (Enable).
  5. اكتب اسم المشروع الذي سيظهر للمستخدمين عند تسجيل الدخول، واختر بريد الدعم الخاص بك.
  6. اضغط على زر Save (حفظ).
الآن أصبح إيميل جوجل الموحد ht.suez.ads@gmail.com مصرحاً له بالدخول الآمن!

الجزء الثالث: إنشاء قاعدة بيانات جوجل (Cloud Firestore)

  1. من القائمة الجانبية اليسرى، اضغط على Build ثم اختر Firestore Database.
  2. اضغط على زر Create database (إنشاء قاعدة بيانات).
  3. اختر موقع الخادم القريب منك (مثل أوروبا) ثم اضغط Next.
  4. اختر البدء في Test Mode (وضع التجربة) لكي تمنح البرنامج صلاحية القراءة والكتابة مباشرة بدون قيود، ثم اضغط Create.
  5. بمجرد اكتمال الإنشاء، اذهب لتبويب Rules في الأعلى وتأكد من كتابة قواعد القراءة والكتابة كالتالي:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

الجزء الرابع: الحصول على مفاتيح الربط ووضعها في البرنامج

  1. في لوحة تحكم Firebase، اضغط على أيقونة الترس ⚙️ بجانب Project Overview واختر Project settings.
  2. انزل لأسفل الصفحة تحت قسم Your apps واضغط على أيقونة الويب </> لتسجيل تطبيقك.
  3. اكتب اسماً للتطبيق (مثل: HT-App) ثم اضغط Register app.
  4. انسخ المفتاحين التاليين فقط من الكود البرمجي الذي سيظهر لك:
    - ApiKey
    - ProjectId
  5. افتح برنامجك وسجل الدخول كـ المدير العام (admin).
  6. اذهب إلى قسم إعدادات النظام والنسخ من شريط التنقل.
  7. تحت قسم إعداد الاتصال السحابي (Firebase)، قم بلصق المفتاحين واضغط حفظ وتطبيق الاتصال.

الجزء الخامس: رفع الاستضافة ونشر الرابط للموظفين مجاناً

  1. قم بزيارة موقع Netlify وسجل حساباً مجانياً.
  2. اذهب لقسم Sites في لوحة تحكم Netlify.
  3. قم بسحب مجلد مشروعك بالكامل billboard-sales-manager وأفلته في الصندوق المخصص للرفع السريع.
  4. سيعطيك الموقع فوراً رابطاً عاماً ومحمياً للموقع (مثل: https://ht-adv-suez.netlify.app).
  5. أرسل الرابط لموظفيك الـ 10 وابدأوا العمل المشترك فوراً ومباشرة عبر الإنترنت!
